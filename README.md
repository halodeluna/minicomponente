# minicomponente
Minicomponente is a multidisciplinary artist from Argentina. 
⡞⠳⣄⣀⣠⠞✿⢷ ֹ۪

She released her first album, "Pierde el ritmo" in 2007, and has since been crafting a personal sonic world where songs navigate between intimate pop, healing frequencies, drone and ambient music, with influences from Butoh dance, vintage technology and anime culture.
